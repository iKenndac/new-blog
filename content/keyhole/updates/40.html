Keyhole 1.1
Improvements
  • Added support for Doppler (thanks to Graham Hall for the contribution!).
  • Added support for Cog.
  • Added a basic AppleScript API for forwarding media commands through to the chosen media player.
  • Minor UI tweaks and improvements.
Bug Fixes
  • Fixed Keyhole not responding to media keys on Apple keyboards (fixed in 1.0.2).
  • The Permission Doctor window should be better behaved interacting with other windows.
  • Fixed an issue that could cause Keyhole to hang while attempting to determine if it has permission to automate an app.
